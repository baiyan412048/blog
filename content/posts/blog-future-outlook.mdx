其實從很早之前就有先建好部落格，只是一直很懶散，沒有正視寫文章這件事，  
我很喜歡無名小站，所以對我來說，應該就是弄一個自己的網誌這件事情很吸引我，  
雖然一直停在樣式、功能等等的地方糾結，寫程式這件事本身還是比較好玩，  
最近真的有下定決心的寫一些東西，但模式應該比較偏向自言自語，  
寫給我自己看，做一點紀錄，順便寫一些工作上的紀錄，不要浪費自己空閒的時間。

## 希望未來可以補上的功能

`task-list` 的樣式應該會需要優先調整，預設的樣式沒有很喜歡，  
部落格樣式應該會邊寫文章邊微調，一些間距跟顏色都還沒跟自己定案，  
先紀錄一些目前有想到的東西 _(2023/10/04)_ ，等之後有想法再陸續補上。

關於及相片集的頁面應該會放到之後再說，相片集只是一個自己的虛榮心，放一些覺得自己拍得不錯或是有紀念價值的照片，  
關於其實好像可有可無，應該會想辦法簡化，畢竟這個部落格比較偏向記錄自己，不需要向自己介紹自己。

- [ ] 相片集頁面 (個人興趣是拍照，希望未來有一個地方可以分享照片 😋)
- [ ] 關於頁面 (需要再整理一下要放什麼內容)
- [ ] 部落格樣式 (還有一些地方需要再微調，設計癖發作)
- [ ] Go top 按鈕
- [ ] 網頁小動態 (參考 [Tailwindcss Blog](https://tailwindcss.com/blog/2023-08-07-meet-studio-our-new-agency-template) / [Motion](https://www.framer.com/motion/))
- [ ] 優化TOP (Table of content)
- [ ] 小字 component
- [ ] 修正 `pre` 卷軸樣式

## 已歸檔的功能

- [x] `Tag cloud` 的功能 (新增文章標籤)

第一個完成是 `Tag cloud` ，主要取代分類這件事情，  
依我的尿性產文章的速度不會快到哪裡去，用 Tag 分類的方式應該會更好找文章，  
目前也沒有規劃搜尋或篩選的功能，等規模出來再說 (: 20篇 ?) ，  
這應該也會當部落格第一篇文章 🌿

- [x] Markdown `task-list` 的樣式調整

當使用 `task-list` 會產 checkbox，但原生的 checkbox 蠻不好看的，所以自己客製了樣式。

```md
- [ ] 測試 1
- [x] 測試 2
```
